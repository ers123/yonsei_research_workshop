# 실습 1: 자료탐색

**핵심 원칙:** 에이전트에게 "무엇을 찾을지"가 아니라 **"어떻게 찾을지의 체계"**를 주기.

---

## Track A — NTIS 기반 국내 동향 파악

### 1단계: NTIS 수동 검색

1. https://www.ntis.go.kr 접속 (로그인)
2. 본인 관심 키워드 검색 (예: "청소년 수면", "스마트홈 에너지")
3. 상위 5개 과제 초록을 복사해 하나의 텍스트 파일로 저장
4. 각 과제의 연구책임자, 과제번호, 수행기관, 총연구비도 수집

### 2단계: Claude/ChatGPT에게 요약 의뢰

```
[역할]
너는 이 분야의 동료 연구자야. 아래 5개 과제 초록을 읽고 답해줘.

[자료]
(과제 5개 초록 붙여넣기)

[요청]
1. 5개 과제에 공통된 연구 흐름을 3-4줄로 정리
2. 과제별 독특한 접근법을 한 줄씩
3. 연구 공백(Research Gap)으로 보이는 주제 2개 제안
4. 각 주장에 어느 과제를 근거로 삼았는지 명시
```

### 3단계: 검증

```
위 답변 중 "공백으로 보이는 주제" 2개에 대해, 
정말 공백인지 해외 문헌을 간단히 확인해줘.
(Perplexity 또는 NotebookLM에 같은 질문을 별도로 던져 교차 검증)
```

**왜 이렇게 하나?** AI 한 번 응답을 정답으로 믿지 않기 위함. 검증 부채(Verification Debt, Kwon 2026) 방지.

---

## Track B — AI 기반 문헌 탐색

### 프롬프트 템플릿

```
[주제]
(예: 한국 청소년의 SNS 사용과 수면 품질 관계)

[요청]
최신 문헌 10개를 찾아줘. 각각:
- 제목
- 저자 (제1저자 + 교신저자)
- 연도
- 학술지
- 핵심 기여 (200자 이내, 본인 분야 박사과정생이 이해할 수준)
- 방법론 한 줄

[특히 다음 관점에서 차이를 강조]
(예: 한국/동아시아 문화 맥락, 또는 종단 vs 횡단, 또는 질적 vs 양적)

[제약]
- 2023-2026 논문 중심
- 각 결과의 DOI 또는 검색 가능한 서지정보 반드시 포함
- 확인 불가한 논문은 만들지 말 것. 모르면 "확인 필요"로 표시.
```

### 검증

AI가 제시한 논문 중 **최소 3개는 본인이 직접 원문 확인**. 환각(hallucination)된 존재하지 않는 논문이 섞여 있을 수 있습니다.
