## ✅ 실습 전 준비

### 공통

- 노트북 충전 100% + 전원 어댑터
- 본인 관심 연구 주제 1개 머리에 담아오기 (Track A 에서 사용)
- Chrome / Firefox / Safari 중 하나

### 권장 — Track B 원활 진행용 (있으면 좋음)

- **Claude Pro 또는 ChatGPT Plus 계정 ($20/월)**
  — 둘 다 없으면 강사 제공 백업 추출본으로 실습 가능
- **Claude Code 설치** — OS 별 공식 설치 명령

  **macOS / Linux**:
  ```bash
  curl -fsSL https://claude.ai/install.sh | bash
  ```
  설치 후 `~/.local/bin` 이 PATH 에 없다는 경고가 나오면 출력된
  `echo 'export PATH="$HOME/.local/bin:$PATH"' >> ~/.zshrc && source ~/.zshrc` 실행.

  **Windows (PowerShell)**:
  ```powershell
  irm https://claude.ai/install.ps1 | iex
  ```
  실행 정책 에러 시: `Set-ExecutionPolicy -Scope CurrentUser RemoteSigned` 1회 실행 후 재시도.

  상세: repo 의 `hands_on/SETUP.md`

### Track C · 원하시는 분만 (당일 실습 가능 여부는 노트북에 따라 다름)

- **Ollama 설치** — https://ollama.com/download
- **본인 RAM 에 맞는 모델 1개** pull (앱이 자동 감지):

  | RAM | 명령 | 크기 |
  |---|---|---|
  | **16GB+** | `ollama pull gemma4:e4b` | ~9.6GB |
  | **8GB** | `ollama pull qwen2.5:3b` | ~2GB |
  | **4GB+** | `ollama pull gemma3:1b` | ~0.8GB |

  > Lite 앱은 설치된 것 중 **가장 큰 지원 모델을 자동 선택**. 하나만 pull 하면 됨.
- Part 3 실습 중 막힌 분은 도와드립니다.

### Track C · 나만의 연구팀

> ⚠️ **2026-04-23 변경**: 기존엔 "홈스터디 전용" 이었으나 **Streamlit Lite 변형** 이 추가돼 **16GB 이상 노트북은 당일 실습 가능** 해졌습니다. 하드웨어가 약한 분은 집에서 차분히 재현하는 쪽이 안전합니다.

**대상**: 로컬 모델로 3-에이전트 파이프라인을 직접 돌려보고 싶은 분

#### 당일 실습 — Streamlit Lite 경로 (16GB 이상 권장)

**macOS / Linux**:
```bash
cd demo/streamlit_research_team_lite
python3 -m venv .venv && source .venv/bin/activate
pip install -r requirements.txt
ollama pull gemma4:e4b        # 처음 한 번만 (9.6GB, 5-15분)
streamlit run app.py
```

**Windows (PowerShell)**:
```powershell
cd demo\streamlit_research_team_lite
python -m venv .venv
.\.venv\Scripts\Activate.ps1
pip install -r requirements.txt
ollama pull gemma4:e4b
streamlit run app.py
```

브라우저가 http://localhost:8501 을 자동으로 엽니다. 사이드바에 **"✓ gemma4:e4b"** 녹색 배지가 뜨면 준비 완료.

## 🚨 실습 중 자주 빠지는 세 가지 함정

### ① 검증부채 (Verification Debt)
AI 가 빠르게 내놓으니 그냥 받고 싶지만, 그게 "검증부채" 입니다.
→ **한 단계 내놓으면 한 단계 확인, 누적 금지.**

### ② 맥락 포화
CLAUDE.md 에 모든 걸 넣으면 에이전트가 길을 잃습니다.
→ **한 페이지만, 자주 쓰는 것만, 부가 정보는 링크로.**

### ③ AI Brain Fry
복잡한 에이전트를 오래 돌리다 자기가 뭘 하고 있었는지 잊어버립니다.
→ **2시간마다(또는 정기적으로) 요약을 파일로 저장, 다음 세션은 그 파일부터 시작.**

## 🧭 로컬 모델의 "민감 자료" 개념 — 두 가지 구분

로컬 모델이 IRB / 프라이버시 를 전부 해결한다는 말은 **절반만 맞습니다**:

| 차원 | 로컬이 해결? |
|---|---|
| **(a) 전송 금지** — 인터뷰 전사 · 미공개 데이터가 외부 API 로 안 나감 | ✅ **해결** (IRB 준수) |
| **(b) 주제 민감성** — 섭식장애·자살·성 등 주제 자체가 민감해서 모델이 거부 | ❌ **해결 안 됨** |

**(b) 걸릴 때 4가지 선택지** (우선순위 순):
1. **프롬프트 프레이밍 변경** — "생성" 이 아니라 "분석·분류·태깅" 태스크로 리프레이밍
2. **Uncensored 변종 사용** — `0xIbra/supergemma4-26b-uncensored-gguf-v2` 등
3. **다른 base 모델** — Mistral 계열이 Gemma 보다 덜 censored
4. **태스크 쪼개기** — 민감 생성은 연구자 · 분석·분류·요약만 LLM

상세: [appendix/04_Uncensored_로컬모델_연구활용_가이드.md](https://github.com/ers123/yonsei_research_workshop/blob/main/appendix/04_Uncensored_로컬모델_연구활용_가이드.md) 2.5절

## 🔧 자주 마주치는 트러블

| 증상 | 해결 |
|---|---|
| Claude Pro 없음 / ChatGPT Plus 없음 | Track B 는 백업 추출본으로 진행 · Track A 는 무료 Claude/Gemini 로 가능 |
| PDF 업로드 실패 | `hands_on/references/ai_index/ai_index_extracted_backup.md` 백업본 사용 |
| `claude: command not found` 설치 직후 (macOS) | `echo 'export PATH="$HOME/.local/bin:$PATH"' >> ~/.zshrc && source ~/.zshrc` 1회 실행 |
| `claude` 명령 인식 안 됨 (Windows) | PowerShell 재시작 · 안 되면 `$env:Path += ";$HOME\.local\bin"` 임시 적용 후 영구 PATH 추가 |
| Ollama 설치가 macOS 12 에서 `brew install` 실패 | 공식 installer (.zip) 사용: https://ollama.com/download/mac |
| `gemma4:e4b` 다운로드가 9GB 넘게 받음 | 정상입니다 — effective 4B MatFormer 구조로 실제 크기 9.6GB |
| Streamlit Lite 사이드바 "미설치" 경고 | 표시된 `ollama pull gemma4:e4b` 명령 실행 후 브라우저 새로고침 |
| 첫 호출이 5분 넘게 걸림 | 정상 (CPU 추론 + 컨텍스트 주입). 두 번째부터 빠름 |
| Windows PowerShell 권한 에러 | `Set-ExecutionPolicy -Scope CurrentUser RemoteSigned` 1회 실행 |
| "같은 명령인데 다르게 동작" | Claude / GPT / Gemini 는 각자 업데이트됨. 재현성 원하면 로컬 모델로 |
