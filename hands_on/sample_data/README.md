## 데이터 구조

각 텍스트 파일은 다음 필드를 포함합니다:

- **제목**: 논문 제목 (한국어)
- **저자**: 가상의 저자명
- **발행연도**: 2022-2025
- **학술지**: 학술지명
- **DOI**: 가상의 DOI
- **초록**: 3-4문장의 한국어 초록
- **주요 연구결과**: 3개 항목
- **연구방법**: 1문장 요약
- **키워드**: 3-4개 핵심어

## 유의사항

- 이 데이터는 **워크숍 실습 목적으로 생성된 가상의 논문**입니다.
- 저자, DOI, 통계 수치 등은 모두 허구이며, 실제 연구와 무관합니다.
- 분류 연습, 구조화 연습, LLM 질의 연습 등에 활용할 수 있습니다.
