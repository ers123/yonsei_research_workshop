# 연구 프로젝트 맥락 문서

## 소속
연세대학교 생활과학대학 대학원 (석사과정)

## 연구 분야
심바이오틱라이프텍연구원 소속 — 노화과학, 식품영양, 섬유패션, 휴먼라이프, 주거복지, 디자인

## 현재 단계
연구과제 선정 전 — 주제 탐색 중

## 요구사항
1. NTIS(ntis.go.kr)에서 관련 국가 R&D 과제를 검색하여 연구 트렌드 파악
2. 최근 3-5년간 주요 선행연구 탐색
3. 연구 갭(gap) 식별
4. Research Question 도출

## 출력 규칙
- 문체: 학술체 (~이다)
- 인용: APA 7th (저자, 연도)
- 언어: 한국어
- 모든 논문 인용은 실제 존재 여부를 명시 (확인됨 / 확인 필요)

## 검증 규칙
- 선행연구 정리 후, 로컬 모델(ollama)로 교차검증 실행
- 검증 실패 항목은 최종 결과에서 제외

## 금기
- 존재하지 않는 논문을 만들지 말 것
- 확실하지 않으면 "확인 필요" 표기
- 추측으로 연구 갭을 만들지 말 것
